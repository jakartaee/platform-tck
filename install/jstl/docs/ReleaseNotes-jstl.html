Jakarta EE Standard Tag Library Technology Compatibility Kit, Version 2.0

Release Notes, May 2021

Kit Contents

A Jakarta EE Standard Tag Library, Version 2.0, Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta EE Standard Tag Library TCK signature, API and specification tests:

    • A signature test that checks that all of the public APIs are supported or defined in the Jakarta EE Standard Tag Library Version 2.0 implementation that is under test.

    • Specification tests for the testable assertions required by the Jakarta EE Standard Tag Library Revision 2.0 Specification.

    • API tests for the public APIs and language elements defined by the Jakarta EE Standard Tag Library 2.0 Specification.

Platform Notes

The Jakarta EE Standard Tag Library 2.0 TCK tests a Jakarta EE Standard Tag Library 2.0 Compatible Implementation (CI) and can be run on JavaTM Platform, Standard Edition (Java SE) version 8 or 11 using a suitable web container in which you run the tests. See Compatible Implementation for Jakarta EE Standard Tag Library 2.0 for additional information.

Compatible Implementation for A Standard Tag Library 2.0

The designated Runtime for compatibility testing of implementations based upon the Jakarta EE Standard Tag Library Specification 2.0 is as follows:
  • This Jakarta EE Standard Tag Library TCK has been built on JDK 8 (1.8) and has been tested with Java Standard Edition 8 and 11.
  • Eclipse Compatible Implementation (CI) of Jakarta EE Standard Tag Library 2.0 available as a standalone implementation from Jakarta EE Specifications web site, or included as part of the Jakarta EE 9.1 compatible implementation, for example Eclipse GlassFish 6 or 6.1, or the Eclipse GlassFish 6 or 6.1 Web Profile compatible implementation.

    Note - Jakarta EE Platform compatibility is not required; this TCK only requires a container that supports Jakarta EE Standard Tag Library 2.0 and its requirements.

  • Jakarta EE Standard Tag Library TCK tests run on the following platforms:
    • Alpine Linux v3.12
    • CentOS Linux 7
