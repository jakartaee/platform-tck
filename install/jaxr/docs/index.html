Guide to JAXR TCK 1.0 Documentation

The JAXR TCK 1.0 documentation includes the following:

  • The JAXR ReleaseNotes provide the information that you need to install, set up, and run the JAXR TCK, Version 1.0.

  • The JAXR TCK User's Guide also provides the information that you need to install, set up, and run the JAXR TCK, Version 1.0. In addition, the guide provides the rules you must comply with to pass the JAXR TCK.

  • The Javadoc Assertion List lists all the javadoc assertions that are tested by the JAXR TCK.

  • The Specification Assertion List lists all the specification assertions that are tested by the JAXR TCK.
