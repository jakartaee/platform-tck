Jakarta RESTful Web Services 3.0 TCK Documentation

The Jakarta RESTful Web Services 3.0 TCK documentation includes the following:

  • The Jakarta RESTful Web Services 3.0 Release Notes, provide late-breaking information about the Jakarta RESTful Web Services 3.0 TCK

  • The Jakarta RESTful Web Services 3.0 Technology Compatibility Kit User's Guide provides the information that you need to install, set up, and run the Jakarta RESTful Web Services 3.0 TCK. The guide also provides the rules with which you must comply to pass the Jakarta RESTful Web Services 3.0 TCK.

  • The Javadoc Assertion List includes the javadoc assertions that are indirectly tested by the Jakarta RESTful Web Services 3.0 TCK.

  • The Specification Assertion List includes the specification assertions that are tested by the Jakarta RESTful Web Services 3.0 TCK.
