Jakarta Connectors Specification Technology Compatibility Kit, Version 1.7
Release Notes, September 2019

Kit Contents

The Jakarta Connectors, Version 1.7 Technology Compatibility Kit (TCK) includes the following items:

  • JCA TCK tests for signature, API, and End-to-End tests:

    • A signature test checks that all of the public APIs are supported in the JCA Version 1.7 implementation that is being tested

    • API tests for all of the public APIs under the javax.resource package (and its sub-packages)

    • End-To-End tests that demonstrate compliance with the Jakarta Connectors 1.7 Specification

Platform Notes

The Jakarta Connectors TCK tests have been built and tested with JavaTM Platform, Standard Edition 8 (Java SE 8).

The Jakarta Connectors TCK tests have been run on the following platforms:

  • MAC OS X, El Capitan (10.9.5+)
  • Oracle Linux 7.1
  • Windows 10

The Jakarta Connectors TCK tests have been run against the JCA 1.7 Compatible Implementation, Eclipse GlassFish 5.1.

Installing, Setting Up, and Running the JCA TCK

Refer to the Jakarta Connectors TCK 1.7 User's Guide for complete instructions on installing, setting up, and running the JCA TCK. The online version of the JT Harness version 5.0 documentation is available here.

JCA TCK Requirements, Facts, and Figures

The JCA TCK requires a minimum of 203 MB of free disk space:

  • JCA TCK (zipped): 31 MB
  • JCA TCK (unzipped): 164 MB
  • Space for JCA log/report files: 8 MB minimum

The test suite bundle contains the following:

  • EARs: 0
  • JARs: 39
  • WARs: 24
  • RARs: 17

  • Total tests: 138

Status Information

Exclude List

You can find the exclude list in the JCA bundle in <TS_HOME>/bin/ts.jtx. Since the list can change over time, please be sure to get the latest exclude list available from the Jakarta Connectors Specification Web Site: https://jakarta-ee/specifications/connectors/1.7/

Known Problems and Workarounds

Running the JCA TCK Against Java EE RI on Windows

When configuring and running the JCA TCK against the Eclipse GlassFish Compatible Implementation (CI) on Windows, please make sure to install all related software (Java SE, the CI, the TCK, etc.) on the same drive (for example, C:). If you want to install on a drive other than C:, please be sure to change the following properties in <TS_HOME>/bin/ts.jte:

s1as.applicationRoot=c:
