Jakarta Connectors Specification Technology Compatibility Kit, Version 2.0
Release Notes, September 2020

Kit Contents

The Jakarta Connectors, Version 2.0 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta Connectors TCK tests for signature, API, and End-to-End tests:

    • A signature test checks that all of the public APIs are supported in the Jakarta Connectors Version 2.0 implementation that is being tested

    • API tests for all of the public APIs under the jakarta.resource package (and its sub-packages)

    • End-To-End tests that demonstrate compliance with the Jakarta Connectors 2.0 Specification

Platform Notes

The Jakarta Connectors TCK tests have been built and tested with JavaTM Platform, Standard Edition 8 (Java SE 8).

The Jakarta Connectors TCK tests have been run on the following platforms:

  • MAC OS X, El Capitan (10.9.5+)
  • Oracle Linux 7.1
  • Windows 10

The Jakarta Connectors TCK tests have been run against the Jakarta Connectors 2.0 Compatible Implementation, Eclipse GlassFish 6.

Installing, Setting Up, and Running the Jakarta Connectors TCK

Refer to the Jakarta Connectors TCK 2.0 User's Guide for complete instructions on installing, setting up, and running the Jakarta Connectors TCK. The online version of the JT Harness version 5.0 documentation is available here.

Jakarta Connectors TCK Requirements, Facts, and Figures

The Jakarta Connectors TCK requires a minimum of 203 MB of free disk space:

  • Jakarta Connectors TCK (zipped): 31 MB
  • Jakarta Connectors TCK (unzipped): 164 MB
  • Space for Jakarta Connectors log/report files: 8 MB minimum

The test suite bundle contains the following:

  • EARs: 0
  • JARs: 39
  • WARs: 24
  • RARs: 17

  • Total tests: 138

Status Information

Exclude List

You can find the exclude list in the Jakarta Connectors bundle in <TS_HOME>/bin/ts.jtx. Revisions to the Exclude List are published as updates to the TCK bundle. You can check the Jakarta Connectors Specification Web Site for updates: https://jakarta.ee/specifications/connectors/2.0/

Known Problems and Workarounds

Running the Jakarta Connectors TCK Against Jakarta EE CI on Windows

When configuring and running the Jakarta Connectors TCK against the Eclipse GlassFish 6 Compatible Implementation (CI) on Windows, please make sure to install all related software (Java SE, the CI, the TCK, etc.) on the same drive (for example, C:). If you want to install on a drive other than C:, please be sure to change the following properties in <TS_HOME>/bin/ts.jte:

s1as.applicationRoot=c:
