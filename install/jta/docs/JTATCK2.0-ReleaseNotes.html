JTA Technology Compatibility Kit
Release Notes, October-2020

Java Transaction API (JTA) 2.0 Support

The following types are no longer part of the JTA 2.0 API, but continue to be part of the javax.transaction.xa package provided by Java SE as it was in JTA 1.3 API version:

  • javax.transaction.xa.XAException
  • javax.transaction.xa.XAResource
  • javax.transaction.xa.Xid

The Signature tests in JTA 2.0 will use /bin/sig-test_se8.map file (run in JDK8) and validate the fact that these classes are not part of the JTA 2.0 API jar.

Kit Contents

The JTA, Version 2.0 Technology Compatibility Kit (TCK) includes the following items:

  • JT Harness documentation: Refer to the JT Harness documentation for latest documents.
  • JTA TCK Signature and API tests:
    • API tests that verify that all public APIs are supported and/or defined as specified in the JTA Version 2.0 implementation under test.
    • A Signature test that checks that all public APIs are supported and/or defined as specified in the JTA Version 2.0 implementation under test.

Platform Notes

The JTA TCK tests have been built and tested with the JakartaEE 9 Compatible Implementation (GlassFish 6.0) Application Server running on the Java platforms. If you are using a different Web server, you will need to make adjustments to the procedures outlined in the JTA 2.0 TCK User's Guide.

The JTA TCK tests have been validated on the following platforms:

  • CentOS Linux 7

Installing, Setting Up, and Running the JTA TCK

Refer to the JTA 2.0 TCK User's Guide for complete instructions on installing, setting up, and running the JTA TCK.
