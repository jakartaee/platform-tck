Guide to Jakarta EE XML Web Services TCK 3.0 Documentation

The Jakarta EE XML Web Services TCK 3.0 documentation includes the following:

  • The Jakarta EE XML Web Services Release Notes provide last-minute information about the Jakarta EE XML Web Services TCK, Version 3.0. The Release Notes are located in the TCK product bundle.

  • The Jakarta EE XML Web Services 3.0 Technology Compatibility Kit User's Guide (also in PDF) provides the information you need to install, configure, and run the Jakarta EE XML Web Services TCK. The User's Guide also contains the rules you need to pass for certification.

  • The Javadoc Assertion List lists all the javadoc test assertions for Jakarta EE XML Web Services TCK.

  • The Specification Assertion List lists all the specification test assertions Jakarta EE XML Web Services TCK.

  • The BasicProfile 1.1 (BP1.1) Specification Assertion List lists all the WSI BP1.1 specification test assertions.

  • The Attachment Profile 1.0 (AP1.0) Specification Assertion List lists all the WSI AP1.0 specification test assertions.

  • The Simple SOAP Binding Profile 1.0 (SSBP1.0) Specification Assertion List lists all the WSI SSBP1.0 specification test assertions.

  • The Web Services Addressing 1.0 - Core Specification Assertion List lists all the WSA Core specification test assertions.

  • The Web Services Addressing 1.0 - SOAP Binding Specification Assertion List lists all the WSA SOAP Binding specification test assertions.

  • The Web Services Addressing 1.0 - Metadata Specification Assertion List lists all the WSA Metadata specification test assertions.
