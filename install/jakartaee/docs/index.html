Guide to the Jakarta EE 9 Platform TCK Documentation

The Jakarta EE 9 Platform TCK documentation includes the following:

  • The Jakarta EE 9 Platform TCK Release Notes provides information about the test harness, the test suite, and the platforms tested.

    The Release Notes may be udpated and released through the Jakarta EE 9 Platform Specification web site: https://jakarta.ee/specifications/platform/9/ or https://jakarta.ee/specifications/webprofile/9/.

  • The JakartaEE 9 TCK User's Guide provides comprehensive user documentation, including instructions for installing and configuring the test suite, and using the JavaTest test harness to execute the tests. The User's Guide also contains the rules you need to pass for certification.

  • The Specification Assertion List lists all the specification assertions that are tested by the Jakarta EE 9 Platform TCK test suite.
