Jakarta WebSocket Technology Compatibility Kit, Version 1.1
Release Notes, September 2019

Kit Contents

The Jakarta WebSocket, Version 1.1 Technology Compatibility Kit (TCK) includes the following items:

  • Jakarta WebSocket TCK tests signature, API, and End-to-End tests:
    • Signature test that checks that all of the public APIs are supported in the Jakarta WebSocket Version 1.1 implementation that is being tested
    • API tests for the public APIs under the jakarta.websocket and jakarta.websocket.server packages
    • Specification tests for testable assertions required by the Jakarta WebSocket 1.1 Specification

Jakarta WebSocket TCK Facts

The test suite bundle contains the following:

  • 664 total tests (663 tests + 1 signature test)

Platform Notes

The Jakarta WebSocket TCK tests have been built and tested with JavaTM Platform, Standard Edition 8 (Java SE 8).

The Jakarta WebSocket TCK tests have been run on the following platforms:

  • MAC OS X, El Capitan (10.9.5+)
  • Oracle Linux 7.1
  • Windows 10

The Jakarta WebSocket TCK tests have been run against the following Jakarta WebSocket 1.1 Compatible Implementations:

  • Eclipse Compatible Implementation for WebSocket Version 1.1

Installing, Setting Up, and Running the JavaTM API for WebSocket TCK

Refer to the Jakarta WebSocket TCK 1.1 User's Guide for complete instructions on installing, setting up, and running the Jakarta WebSocket TCK. The online version of the JT Harness version 5.0 documentation is available here.
